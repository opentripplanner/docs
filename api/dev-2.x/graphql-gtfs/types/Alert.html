Alert
Object

Alert of a current or upcoming disruption in public transportation

Fields

#

id

ID !
Non-null

Global object ID provided by Relay. This value can be used to refetch this object using node query.

alertHash

Int

hashcode from the original GTFS-RT alert

feed

String

The feed in which this alert was published

agency

Agency
Alert can have multiple affected entities now instead of there being duplicate alerts for different entities. This will return only one of the affected agencies. Use entities instead.

Agency affected by the disruption. Note that this value is present only if the disruption has an effect on all operations of the agency (e.g. in case of a strike).

route

Route
Alert can have multiple affected entities now instead of there being duplicate alerts for different entities. This will return only one of the affected routes. Use entities instead.

Route affected by the disruption

trip

Trip
Alert can have multiple affected entities now instead of there being duplicate alerts for different entities. This will return only one of the affected trips. Use entities instead.

Trip affected by the disruption

stop

Stop
Alert can have multiple affected entities now instead of there being duplicate alerts for different entities. This will return only one of the affected stops. Use entities instead.

Stop affected by the disruption

patterns

[ Pattern ]
This will always return an empty list. Use entities instead.

Patterns affected by the disruption

alertHeaderText

String

Header of the alert, if available

alertHeaderTextTranslations

[ TranslatedString !]!
Non-null

Header of the alert in all different available languages

alertDescriptionText

String !
Non-null

Long description of the alert

alertDescriptionTextTranslations

[ TranslatedString !]!
Non-null

Long descriptions of the alert in all different available languages

alertUrl

String

Url with more information

alertUrlTranslations

[ TranslatedString !]!
Non-null

Url with more information in all different available languages

alertEffect

AlertEffectType

Alert effect

alertCause

AlertCauseType

Alert cause

alertSeverityLevel

AlertSeverityLevelType

Alert severity level

effectiveStartDate

Long

Time when this alert comes into effect. Format: Unix timestamp in seconds

effectiveEndDate

Long

Time when this alert is not in effect anymore. Format: Unix timestamp in seconds

entities

[ AlertEntity ]

Entities affected by the disruption.
